# 🌦️ Weather App (PyQt5)

## 🇧🇷 Português

Aplicação desktop desenvolvida em Python utilizando PyQt5 que permite consultar informações de clima em tempo real a partir de uma cidade informada pelo usuário.

### 🚀 Funcionalidades

* Busca de clima por cidade

* Exibição de temperatura atual

* Descrição das condições climáticas

* Interface gráfica simples e intuitiva

* Uso de emojis para representar o clima

### 🛠️ Tecnologias utilizadas

* Python
* PyQt5
* Requests
* API REST (OpenWeather)

### ⚠️ Configuração da API Key

Este projeto utiliza a API do OpenWeather para obter os dados climáticos.

Por motivos de segurança, a API Key não está incluída no código.

Para utilizar a aplicação, você deve:

1. Criar uma conta em: https://openweathermap.org/

2. Gerar sua própria API Key

3. Substituir no código:

```python
api_key = "SUA_API_KEY_AQUI"
```

### ▶️ Como executar

Instale as dependências:

```bash
pip install PyQt5 requests
```

Execute o projeto:

```bash
python weather-app.py
```

### 💡 Observações

* Certifique-se de estar conectado à internet

* O nome da cidade deve ser válido para obter os dados corretamente

### 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar:

* Desenvolvimento de interfaces gráficas com PyQt5

* Consumo de APIs REST

* Manipulação de eventos e layouts

* Integração com ambiente Linux

## 🇺🇸 English

Desktop application developed in Python using PyQt5 that allows users to fetch real-time weather information based on a given city.

### 🚀 Features

* Search weather by city

* Display current temperature

* Show weather conditions description

* Simple and intuitive graphical interface

* Use of emojis to represent weather conditions

### 🛠️ Technologies

* Python
* PyQt5
* Requests
* REST API (OpenWeather)

### ⚠️ API Key Configuration

This project uses the OpenWeather API to retrieve weather data.

For security reasons, the API key is not included in the code.

To run the application, you need to:

1. Create an account at: https://openweathermap.org/

2. Generate your own API key

3. Replace it in the code:

```python
api_key = "YOUR_API_KEY_HERE"
```

### ▶️ How to run

Install dependencies:

```bash
pip install PyQt5 requests
```

Run the application:

```bash
python weather-app.py
```

### 💡 Notes

* Make sure you are connected to the internet

* The city name must be valid to retrieve data correctly

### 📌 Purpose

This project was developed to practice:

* Building graphical interfaces with PyQt5

* Consuming REST APIs

* Event handling and layout management

* Working with Linux environments
